About This Project

Welcome to Dimensional Playground - an academic exploration into the world of WebVR and browser-based 3D graphics. This project was created to demonstrate modern web animation techniques and interactive 3D development principles.

Built using A-Frame, a powerful WebVR framework, this experience showcases real-time 3D rendering, procedural animations, physics simulations, and dynamic audio feedback powered by the Web Audio API - all running natively in your browser without plugins.

The scene features various geometric primitives including spheres, cubes, cones, tori, dodecahedrons, and icosahedrons - each with unique animation behaviors and interactive properties. Multiple orbiting systems, particle effects, and a dynamic lighting setup create an immersive environment.

Key learning outcomes include understanding 3D coordinate systems, animation easing functions, event-driven interactions, and creating accessible VR experiences that work across desktop and mobile platforms.
