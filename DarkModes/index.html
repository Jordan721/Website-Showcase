About This Project

This is a mockup demonstration of the dark mode feature I developed during my internship with Unadat, a financial wellness portal, during the pandemic. The mockup recreates the functionality and design patterns implemented for the actual Unadat portal.

The Experience:

Despite working remotely throughout the entire internship, I collaborated closely with the CEO and creator of Unadat to lead the development of a comprehensive dark mode feature. I optimized mobile UX, enhanced accessibility, and improved user comfort across devices.

Through user research and A/B testing, I validated design decisions that resulted in measurable improvements in user satisfaction and engagement. Working with design and development teams, I refined the interface, streamlined navigation, and ensured visual consistency while implementing responsive design principles and performance optimizations to improve load times on mobile platforms.

Key Contributions:

  • Led dark mode development with customizable color schemes for accessibility
  • Conducted user research and A/B testing to validate design decisions
  • Optimized mobile UX with responsive design and performance improvements
  • Collaborated with cross-functional teams on feature specifications and sprint planning
  • Implemented smooth theme transitions and persistent user preferences

Technologies Used:

Pure HTML, CSS (CSS Custom Properties), and Vanilla JavaScript - no frameworks required!
