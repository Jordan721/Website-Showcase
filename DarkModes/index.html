About This Project

This dark mode showcase was created as part of my work on Unadat, a website/company that helps people get their finances together.

The Internship Experience:

This project was developed during a SYEP (Summer Youth Employment Program) internship that took place during the pandemic. Despite being completely virtual, there was incredible Camaraderie among the team. Working remotely taught me valuable collaboration skills and how to build meaningful connections even through screens.

One of the most memorable parts of this experience was collaborating with the Creative/Art Intern, who played a crucial role in helping me figure out the color scheme for the dark mode. Their expertise in color theory and visual design was instrumental in creating a palette that was both aesthetically pleasing and accessible for users managing financial data during extended sessions, especially in low-light environments.

Key Features Developed:

  • Multiple toggle implementation methods to explore different UX approaches
  • Customizable color schemes for accessibility and brand consistency
  • Smooth transitions between light and dark modes
  • Persistent theme preferences using localStorage
  • System preference detection for automatic theme selection

Technologies Used:

Pure HTML, CSS (CSS Custom Properties), and Vanilla JavaScript - no frameworks required!
