About This Project

This was my final project for class, created to demonstrate interactive user experiences in 3D space.

Project Goals

The assignment was to create something that users could interact with in meaningful ways. This interactive orbital visualization is what I came up with - combining 3D graphics, responsive controls, and immersive audio.

Arduino Integration

This project was originally designed to work with an Arduino microcontroller using physical potentiometers (rotary knobs and dials). The Arduino would read the analog input from these physical controls and send the data through a serial port to control the 3D camera movement.

However, since most users don't have access to an Arduino and the necessary hardware setup, I adapted the project to use mouse and keyboard controls as an alternative input method. This way, everyone can experience and interact with the visualization!

How It Works

The code still maintains the serial port functionality for those who do have Arduino hardware. If you have an Arduino with two potentiometers connected to analog inputs, you can use them to control the orbital camera by adjusting the physical knobs and dials.
