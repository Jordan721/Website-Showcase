About This Project

This was a project for the Year Up App Dev Bootcamp where I had to create a library management system for a neighborhood using Java.

Original Java Features:

  • View available books in the library
  • Check out books by entering your name
  • View checked out books
  • Return books to the library

Enhanced Web Features:

  • 3D Bookshelf: Interactive books with pull-out animation
  • Search System: Search by title, ID, ISBN, or person name
  • Genre Filters: Browse books by 8 different genres (Classic, Fantasy, Dystopian, Romance, Young Adult, Adventure, Historical)
  • Check In/Out: Click any book to check it out or return it
  • Stats Dashboard: View library statistics and personalized recommendations
  • Donate Books: Add new books with ISBN and genre information
  • Persistent Storage: Your data saves automatically

Each book has: A unique ID number, 13-digit ISBN code, title, author, and genre classification. You can search, filter, check out, and check in books - all with a beautiful library aesthetic!
