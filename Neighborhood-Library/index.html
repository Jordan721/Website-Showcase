About This Project

This was a project for the Year Up App Dev Bootcamp where I had to create a library management system for a neighborhood using Java.

The original Java console application allowed users to:

  • View available books in the library
  • Check out books by entering their name
  • View checked out books
  • Return books to the library

I transformed the console-based Java application into this interactive web experience, maintaining all the core functionality while adding a beautiful, user-friendly interface.
