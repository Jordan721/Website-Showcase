About This Project

This is an enhanced mockup demonstration of the dark mode feature I developed during my internship with Unadat, a financial wellness portal, during the pandemic. The mockup recreates and expands upon the functionality and design patterns implemented for the actual Unadat portal, with additional interactive features to showcase advanced UX capabilities.

The Experience:

Despite working remotely throughout the entire internship, I collaborated closely with the CEO and creator of Unadat to lead the development of a comprehensive dark mode feature. I optimized mobile UX, enhanced accessibility, and improved user comfort across devices.

Through user research and A/B testing, I validated design decisions that resulted in measurable improvements in user satisfaction and engagement. Working with design and development teams, I refined the interface, streamlined navigation, and ensured visual consistency while implementing responsive design principles and performance optimizations to improve load times on mobile platforms.

Key Contributions:

  • Led dark mode development with customizable color schemes for accessibility
  • Conducted user research and A/B testing to validate design decisions
  • Optimized mobile UX with responsive design and performance improvements
  • Collaborated with cross-functional teams on feature specifications and sprint planning
  • Implemented smooth theme transitions and persistent user preferences

Enhanced Features in This Demo:

This showcase goes beyond the original implementation with advanced interactive features:

  • Animated Transitions: Ripple and wave effects that emanate from toggle points for visual feedback
  • Live Code Editor: Real-time CSS variable display with syntax highlighting showing changes as they happen
  • Before/After Comparison: Interactive slider to compare light and dark modes side-by-side (desktop and mobile)
  • Gamification System: 8 unlockable achievements with notification banners and progress tracking
  • Stats Dashboard: Real-time counters for theme toggles, color changes, and achievements
  • Theme Sharing: Generate unique URLs to share custom color schemes with others
  • Theme History: Visual timeline of up to 12 previous themes with one-click restoration
  • Auto-Scheduling: Time-based automatic switching between light and dark modes
  • Advanced Interactions: Double-click background to toggle, keyboard shortcuts (Ctrl/Cmd+Shift+D)
  • Persistent Storage: All preferences, stats, and history saved locally for seamless experience

Technologies Used:

Pure HTML, CSS (CSS Custom Properties with advanced animations), and Vanilla JavaScript (ES6+) - no frameworks required! Features include:

  • CSS Grid & Flexbox for responsive layouts
  • CSS Custom Properties for dynamic theming
  • LocalStorage API for data persistence
  • URL Search Params for theme sharing
  • Touch events for mobile slider support
  • Keyframe animations and transitions

Design Principles:

  • Progressive Enhancement: Core functionality works without JavaScript
  • Mobile-First: Fully responsive with touch-optimized interactions
  • Accessibility: Keyboard navigation, semantic HTML, ARIA labels
  • Performance: Optimized animations, debounced events, efficient DOM updates
  • User Experience: Immediate feedback, smooth transitions, intuitive controls
