About This Project

This was a project for the Year Up App Dev Bootcamp where I had to create a library management system for a neighborhood using Java.

Original Java Features:

  • View available books in the library
  • Check out books by entering your name
  • View checked out books
  • Return books to the library

Enhanced Web Features:

  • Warm Library Theme: Cozy, traditional library aesthetic with rich wooden browns, warm tans, and elegant parchment colors - designed to feel like stepping into a classic neighborhood library
  • Wide Bookshelf Display: Books displayed as colorful spines on a realistic wooden bookshelf with grain texture - hover to lift a book and reveal its information with smooth bouncy animations
  • Interactive Hover Cards: When you hover over a book spine, it lifts up with a satisfying animation and displays a detailed info window showing title, genre, ID, ISBN, and availability status
  • Custom Cursors: Interactive pointer hand (ðŸ‘†) when hovering over books and tapping finger (ðŸ‘‡) when clicking - best experienced on desktop
  • Search System: Search by title, ID, ISBN, or person name with elegant, responsive input fields
  • Genre Filters: Browse books by 8 different genres with beautiful ripple effect buttons (Classic, Fantasy, Dystopian, Romance, Young Adult, Adventure, Historical)
  • Check In/Out: Click any book to check it out or return it through polished modal windows
  • Stats Dashboard: View library statistics and personalized recommendations in a beautifully styled dashboard
  • Donate Books: Add new books with ISBN and genre information through an elegant form interface
  • Admin Mode: Password-protected access (password: library2025) to permanently remove books from the library inventory
  • Persistent Storage: Your data saves automatically using browser local storage
  • Delightful Animations: Smooth transitions, bouncy button effects, glowing hovers, and professional micro-interactions throughout

Each book has: A unique ID number, 13-digit ISBN code, title, author, and genre classification. Browse the wide bookshelf, hover over book spines to see their details, and click to check out - all with a beautiful warm library aesthetic featuring rich wooden textures, soft lighting, and that cozy bookshop atmosphere!
