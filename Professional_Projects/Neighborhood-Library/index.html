About This Project

This was a project for the Year Up App Dev Bootcamp where I created a library management system using Java.

Original Java Features

  • View available books in the library
  • Check out books by entering your name
  • View checked out books
  • Return books to the library

Enhanced Web Features

  • Dark Academia Theme: Elegant, moody aesthetic with gold accents and vintage typography
  • Animated Particles: Floating golden dust particles creating an magical atmosphere
  • 3D Book Spines: Interactive bookshelf with lift animations on hover
  • Flickering Candles: Animated candle flames in the header
  • Search & Filter: Find books by title, ISBN, or borrower name
  • Genre Categories: Browse by 8 different literary genres
  • Stats Dashboard: View library analytics and recommendations
  • Donate System: Add new books to the collection
  • Curator Mode: Admin access (password: library2025) to manage inventory
  • Persistent Storage: Data saves automatically to browser storage
